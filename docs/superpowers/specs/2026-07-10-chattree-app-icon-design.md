# ChatTree 应用图标设计

## 方向

采用用户选定的 C「知识树」方案。图标以低饱和青绿色圆角方形为底，雾白色树干连接四个知识节点；其中两个节点使用石墨色，两个节点使用暖珊瑚色，呼应当前浅色石墨工作台的中性色与轻量强调色。

## 交付

- 保留 `Assets/ChatTree.svg` 作为可编辑源文件。
- 生成包含 16、24、32、48、64、128、256 像素图层的 `Assets/ChatTree.ico`。
- 将 ICO 设为 WPF 可执行文件图标和主窗口图标。
- 不替换现有侧栏 Emoji 图标，也不改变任何业务功能。

## 验证

- 自动测试确认 ICO 被打包为 WPF 资源。
- Debug 与 Release 构建成功。
- 启动真实窗口后确认窗口句柄、标题、响应状态及图标资源加载正常。
